## To-Do-List
This is an assignmnet done for NUS Computing for Voluntary Welfare Organisations (CVWO) application in 2020.

### Features

- User authentication
    - Register / Login / Forgot Password
- Todo-lists
    - CRUD operation
    - Search by any txt


### Setup
- Software used: Rails, Ruby, Node
- CLI setup
`$ yarn install`
`$ bundle install`
`$ rails db: create`
`$ rails db: migrate`
run: `$ rails: server`
